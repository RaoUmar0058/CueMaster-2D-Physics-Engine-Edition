## 🚀 Overview

**CueMaster 2D** is a lightweight, high-performance pool/carrom simulation built completely from scratch using **Vanilla JavaScript** and **HTML5 Canvas**. 

Instead of relying on open-source physics libraries (like Matter.js or P5.js), this project features a **natively coded rigid-body physics engine**. It showcases advanced rendering architecture, matrix coordinate manipulation, and rigorous vector mathematics to handle mass and velocity transformations in real time.

## 🛠️ Core Engineering Highlights

* **Custom Vector Math Pipeline:** Handles acceleration, direction tracking, and dynamic continuous drag interactions without framework abstractions.
* **Elastic Boundary Transformations:** Wall/cushion collisions are evaluated using pure kinetic inversion matrices (`vx *= -1`), preserving structural momentum stability.
* **Realistic Friction Co-efficient:** Implements smooth multi-frame exponential velocity dampening to accurately simulate real felt-table resistances.
* **Real-time Trajectory Mapping:** Displays dynamic visual pre-shot guidelines using dynamic vector projecting dashes.
